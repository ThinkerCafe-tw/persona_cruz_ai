## 🎯 軍團協作模式

### 日常協作
- **晨會**：各軍團統帥向太極宇宙觀察者匯報
- **作戰**：根據任務性質，組建跨軍團特遣隊
- **覆盤**：每日結束時進行戰果總結

### 特殊編隊
1. **快速反應部隊**：電火 + 劍金 + 露水
